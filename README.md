# Live Commenting System

## Objective
- Create a live commenting system where users can add, edit, and delete comments for a blog post. Other users should see these changes in real-time.

## Features
- Turbo Streams for real-time updates to comments across different users (add, update, delete).
- Stimulus controllers to handle features like toggling comment forms, character counters for input fields, and AJAX-based form submission.
- Use Turbo Frames for comment rendering and inline editing.
